# Claw 导航

> AI Agent 导航与对比网站 — 一站式比较各类 AI 编程助手

## 功能

- 📋 **Agent 导航**：收录 10 款主流 AI 编程 Agent（GitHub Copilot、Cursor、Windsurf、Claude、Codeium 等）
- 🔍 **搜索 & 筛选**：按名称、功能或分类快速过滤
- ⚖️ **横向对比**：最多同时对比 4 款 Agent，多维度对比表格
- 📊 **详情页**：功能特性、定价方案、优劣势分析一览无余
- 🌙 **暗色主题**：护眼暗色设计，响应式布局

## 使用方法

直接用浏览器打开 `index.html` 即可，无需构建或安装依赖。

## 文件结构

```
.
├── index.html    # 主页面 HTML
├── styles.css    # 样式表
├── agents.js     # Agent 数据
└── main.js       # 交互逻辑
```
