## 一、项目概述

| 属性 | 值 |
|------|-----|
| 项目名称 | Claw Navigator（龙虾导航） |
| 项目定位 | 中文世界首个 Claw/龙虾 AI Agent 生态导航与对比平台 |
| 工作目录 | `C:\Users\Admin\WorkBuddy\claw-navigator\` |
| 技术栈 | Astro 5.x + Tailwind CSS 4.x + React 19.x（Islands） |
| 包管理器 | pnpm |
| Node版本 | >= 18（推荐20+） |
| 部署目标 | 静态托管（Vercel / Netlify / GitHub Pages） |
| 开发周期 | Phase 1 MVP: 1-2天 |
