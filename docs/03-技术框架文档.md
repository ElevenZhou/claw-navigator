## 三、Astro 配置

### astro.config.mjs

```javascript
import { defineConfig } from 'astro/config';
import react from '@astrojs/react';
import tailwind from '@astrojs/tailwind';
import sitemap from '@astrojs/sitemap';

export default defineConfig({
  site: 'https://clawnav.com',  // 正式域名，后续替换

  integrations: [
    react(),
    tailwind(),
    sitemap({
      filter: (page) => !page.includes('/product/compare'),
    }),
  ],

  // 构建配置
  build: {
    format: 'directory',   // /product/openclaw/index.html 风格
  },

  // 压缩
  compressHTML: true,

  // 图片优化
  image: {
    domains: [],
  },

  // Vite 配置
  vite: {
    ssr: {
      noExternal: ['lucide-react'],
    },
  },
});
```

## 六、数据架构

### 6.1 产品数据流

```
products.json (源头)
    ↓ getStaticPaths()
所有产品详情页 (构建时生成)
    ↓ import
首页卡片 / 对比页 / 生态图
    ↓ localStorage
用户对比选中状态
```

### 6.2 数据更新流程

```
1. 编辑 src/data/products.json
2. 运行 pnpm build
3. 部署 dist/ 目录
```

无需数据库、无需后台、无需重启服务。

### 6.3 数据校验规则

| 字段 | 必填 | 类型 | 约束 |
|------|------|------|------|
| id | ✅ | string | 唯一，URL安全字符 |
| name | ✅ | string | ≤50字符 |
| vendor | ✅ | string | 非空 |
| vendorGroup | ✅ | VendorGroup枚举 | 枚举值 |
| slogan | ✅ | string | ≤20字符 |
| description | ✅ | string | 200-300字 |
| platforms | ✅ | Platform[] | 至少1项 |
| deployType | ✅ | DeployType枚举 | 枚举值 |
| price | ✅ | string | 非空 |
| priceCategory | ✅ | PriceCategory枚举 | 枚举值 |
| difficulty | ✅ | 1-5 | 整数 |
| tags | ✅ | string[] | 至少1项 |
| website | ✅ | string | 合法URL |
| color | ✅ | string | HEX格式 #XXXXXX |
| status | ✅ | Status枚举 | 枚举值 |
| github | ⬜ | string/null | 合法URL或null |

## 七、性能目标与策略

### 7.1 目标

| 指标 | 目标值 | 测量工具 |
|------|--------|---------|
| Lighthouse Performance | ≥ 95 | Chrome DevTools |
| Lighthouse SEO | ≥ 95 | Chrome DevTools |
| First Contentful Paint (FCP) | < 1.5s | Lighthouse |
| Largest Contentful Paint (LCP) | < 2.5s | Lighthouse |
| Cumulative Layout Shift (CLS) | < 0.1 | Lighthouse |
| Total JS Size (首屏) | < 50KB | Network面板 |
| 总页面大小（首页） | < 200KB | Network面板 |

### 7.2 策略

| 策略 | 实现方式 |
|------|---------|
| 零JS首屏 | Astro 默认行为，非Island组件不输出JS |
| 按需加载JS | React Islands 使用 `client:visible` / `client:idle` |
| CSS按需 | Tailwind purge，只保留使用到的类 |
| 字体优化 | `font-display: swap` + 预加载关键字体 |
| 图片优化 | Astro `<Image>` 组件，自动 WebP/AVIF + 懒加载 |
| 预连接 | `<link rel="preconnect">` 外部域名 |

## 八、安全规范

| 风险 | 防护措施 |
|------|---------|
| XSS | 搜索输入做 HTML 转义，不使用 innerHTML |
| CSRF | 纯静态站，无状态修改，天然免疫 |
| 数据泄露 | 无用户数据存储，无API Key暴露 |
| 依赖安全 | 定期 `pnpm audit`，及时更新有漏洞的依赖 |
| 内容注入 | 产品数据仅由开发者通过JSON维护，不接受外部输入 |
