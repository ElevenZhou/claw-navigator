## 一、全局最高优先级指令（AI必须100%严格遵守，不得修改/违反）

### AI角色与开发红线
1. AI角色定位：资深全栈开发工程师+产品经理，严格按照这份PRD文档1:1开发「虾塘Hub」网站
2. 绝对不可违反的业务红线：
   - 不是纯技术参数工具站，是「一站式服务门户」
   - 必须遵循「养虾」拟物化主题
   - 全站固定展示「找塘主」微信二维码转化入口
   - 必须100%覆盖SEO功能
   - MVP版本禁止开发在线支付，统一跳转淘宝/闲鱼

### 定死技术栈
- 前端：Vue3 + Element Plus + 响应式布局
- 后端：Node.js + Express + MySQL 5.7+
- 部署：Linux + Nginx
- 兼容：PC+移动端，Chrome/Edge/Safari/Firefox

## 三、产品核心锚点

### 可量化业务目标
- 3个月核心关键词进百度首页，日均UV≥10000
- 访客→微信私域转化率≥5%，服务下单转化率≥1%
- 1个月收录≥50款虾苗
- 打造Claw赛道第一垂直门户

### 用户分层
- 新手小白（65%）：零基础适配、一键安装、报错解决
- 商业落地（25%）：场景化教程、自动化模板、托管服务
- 进阶开发者（10%）：开源收录、参数对比、二次开发

### 核心差异化壁垒
1. 全品类覆盖（4大分类）
2. 全链路闭环（选虾→开塘→养虾→托管）
3. 强主题记忆点（养虾拟物化）
4. 强业务转化（每个页面有转化入口）
5. 新手友好（零技术门槛）

## 五、核心功能需求详情

### 首页（page_home）
1. **全局导航栏**：Logo + 6个导航项 + 搜索框 + 「找塘主」按钮
2. **首屏Banner**：固定文案 + 3个行动按钮
3. **四大分类入口**：原生种虾/品牌成品虾/优选改良虾/平替同科虾
4. **新手首选TOP5**：卡片流，PC端5列/移动2列
5. **教程+资讯**：左3篇教程 + 右3篇快讯
6. **全局Footer**：Logo+Slogan+二维码+快捷导航+服务支持+版权

### 虾苗大全（page_shrimp_list）
1. **分类筛选**：左侧4个一级分类 + 顶部7组二级筛选标签
2. **卡片列表**：PC端4列/移动2列，每卡含详情/加入PK/一键代养
3. **PK池悬浮栏**：右下角固定，最多5款，≥2款可PK

### 虾苗详情（page_shrimp_detail）
1. **核心信息卡**：名称/厂商/更新时间/开源协议/官方链接
2. **测评信息**：亮点+短板、适配/不适配人群、硬件要求、难度评级
3. **塘主实机测评报告**：安装/稳定性/功能实测/避坑提醒
4. **配套养虾手册**：关联教程卡片
5. **用户评价区**（P1）

### 虾苗PK台（page_pk）
1. **自选PK**：搜索勾选2-5款虾苗，生成对比表（8个维度）
2. **预设PK榜单**：4个固定榜单（新手TOP5/微信塘/企业养殖/低配置）
3. **导出图片/分享链接**
4. **每周龙虎榜**（P1）：热度TOP10/好评TOP5/避坑黑名单

### 养虾学堂（page_guide_list/detail）
- 6个分类：新手开塘/日常养虾/调优喂料/虾病防治/塘口玩法/高手养虾经
- 卡片流展示，支持搜索筛选
- 详情页末尾固定转化模块

### 养虾托管中心（page_service_list/detail）
- 6个分类：开塘安装/托管代养/教学培训/成品虾塘/云端虾棚/定制服务
- 下单跳转淘宝/闲鱼，禁止在线支付

### 我的虾塘（page_user）
- MVP简化：收藏/PK记录/订单，手机号验证码登录

## 七、非功能需求

### SEO
- 全站自定义TDK
- URL伪静态格式（.html）
- 图片alt标签+压缩
- sitemap.xml自动生成+推送百度/搜狗
- H1-H6层级规范
- 面包屑导航

### 性能
- PC首屏≤2秒，移动首屏≤3秒，LCP≤2.5秒
- 图片懒加载，CDN接入
- 代码压缩

### 安全
- 后台双因素认证
- 防SQL注入/XSS
- 文件上传校验
- 数据加密存储
- HTTPS强制
- 备案号预留位

### 数据统计
- 百度统计接入
- 页面访问量/停留时长/按钮点击/转化路径
- 虾苗访问量/PK次数/收藏量
